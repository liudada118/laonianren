## 已发现的 Bug 汇总

### 功能 Bug
1. **报告数据格式不匹配** (高): 通过 API 保存的报告数据无法在报告查看页面正确展示
2. **设备在线状态延迟** (中): 一键连接后设备在线状态不能立即反映，需要等待数据帧解析

### UX Bug
3. **评估卡片设备状态不更新** (中): Dashboard 评估卡片右上角的设备在线状态指示器始终显示 0/N
4. **握力页面显示"未知"** (低): 握力评估页面右上角显示"未知"而非设备名称
5. **连接状态反馈延迟** (中): 用户点击一键连接后，按钮立即变为"已连接"，但设备在线指示器仍为 0/7，造成信息矛盾
