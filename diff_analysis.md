# 左右截图差异分析

## 左图（当前实现 - Canvas 渲染）vs 右图（目标效果 - 后端 matplotlib 图片）

### 1. 压力演变图 (Foot Pressure Evolution)

| 差异项 | 左图（当前） | 右图（目标） |
|-------|------------|------------|
| 背景色 | 白色背景 | 灰白/浅色背景 |
| 每帧大小 | 较小，帧之间紧凑 | 较大，每帧有充足空间，足印清晰可辨 |
| 热力图质量 | 像素感明显，颜色不够平滑 | 平滑渐变，高斯模糊效果，颜色过渡自然 |
| 帧标签位置 | 标签在帧上方，紧凑 | 标签在帧上方，有两行（Peak 单独一行） |
| 行标签 | "左脚"/"右脚" 中文 | "Left Foot"/"Right Foot" 英文 |
| 标题 | 中文"足底压力演变（落地→离地）" | 英文 "Foot Pressure Evolution (Start -> End)" |
| 帧背景 | 黑色 | 黑色（一致） |
| 整体尺寸 | 偏小 | 更大更宽敞 |

### 2. 步态平均摘要 (Gait Average Summary)

| 差异项 | 左图（当前） | 右图（目标） |
|-------|------------|------------|
| 整体尺寸 | 较小(560x360) | 更大，足印占据更多空间 |
| 热力图质量 | 像素化明显 | 平滑渐变，高斯模糊，颜色饱满 |
| COP轨迹 | 单条白线 | 多条白色COP轨迹线（每步一条） |
| 标签文字 | 中文"左脚平均"/"右脚平均" | 英文 "Left Foot Average" / "Right Foot Average" |
| 步数标签 | "(共N步)" | "(n=N steps)" |
| 标题 | 中文"步态平均摘要（平滑处理）" | 英文 "Gait Average Summary (Smoothed)" |
| 起止点标记 | 白色圆+红色× | 白色圆圈+红色圆圈（起点白边红心，终点红×） |
| 足印纵横比 | 偏扁 | 更接近真实足印比例，纵向更长 |

### 核心问题总结

1. **热力图平滑度不够**：当前 Canvas 渲染缺少高斯模糊平滑，需要在渲染后对 Canvas 做模糊处理
2. **尺寸太小**：需要增大 Canvas 尺寸和每帧的绘制空间
3. **文字标签**：需要改为英文或保持与后端一致的中文
4. **步态平均的足印比例**：需要更好地保持纵横比
